# PDA端原型页面说明

## 概述
本文件夹包含仓储管理系统的PDA端原型页面，专为移动设备扫描操作优化设计。

## 功能模块

### 1. 首页 (index.html)
- 功能菜单导航
- 用户信息显示
- 设备信息显示

### 2. 入库作业 (入库作业.html)
**主要功能：**
- 扫描容器编码
- 扫描物料标签码
- 输入入库数量
- 查看推荐存储物料
- 标记满托
- 选择入库库位

**操作流程：**
1. 扫描容器编码
2. 查看推荐存储物料
3. 扫描物料编码并输入数量
4. 添加多个物料
5. 确认组盘
6. 选择入库库位
7. 完成入库

### 3. 出库作业 (出库作业.html)
**主要功能：**
- 查看出库单信息
- 扫描容器编码
- 扫描物料定位
- 确认出库数量
- 选择回库库位

**操作流程：**
1. 查看待出库单信息
2. 扫描到达出库口的容器
3. 查看容器物料明细
4. 可选：扫描物料编码快速定位
5. 逐个确认物料出库
6. 全部出库完成后选择回库库位
7. 完成出库

### 4. 盘点作业 (盘点作业.html)
**主要功能：**
- 查看盘点计划
- 扫描容器编码
- 扫描物料定位
- 录入盘点数量
- 添加新物料
- 查看盘点差异

**操作流程：**
1. 查看当前盘点计划
2. 扫描容器编码
3. 查看容器物料明细
4. 可选：扫描物料编码快速定位
5. 逐个录入盘点数量
6. 可选：添加账面没有的物料
7. 完成盘点并选择回库库位

### 5. 库存查询 (库存查询.html)
**主要功能：**
- 按物料查询：查看物料总库存及分布
- 按容器查询：查看容器信息及存储物料
- 按库位查询：查看库位状态及存储物料

**查询方式：**
1. **按物料查询**
   - 扫描或输入物料编码
   - 显示总库存数量
   - 显示库存分布（库位、容器、数量）

2. **按容器查询**
   - 扫描或输入容器编码
   - 显示容器类型、所在库位、状态
   - 显示容器内物料明细

3. **按库位查询**
   - 输入库位编码（排-列-层-深）
   - 显示库位状态、容器信息
   - 显示库位内物料明细

## 设计特点

### 1. 移动端优化
- 响应式布局，适配各种PDA设备
- 大按钮设计，方便点击
- 大字体显示，易于阅读
- 禁止缩放，保持界面稳定

### 2. 扫描优化
- 输入框自动聚焦
- 支持回车键快速提交
- 一键清空输入
- 扫描后自动清空输入框

### 3. 视觉反馈
- 渐变色主题，美观现代
- 卡片式布局，信息清晰
- 状态标签，一目了然
- 高亮显示，快速定位

### 4. 操作便捷
- 流程化操作，步骤清晰
- 实时反馈，即时响应
- 弹窗确认，防止误操作
- 快捷按钮，提高效率

## 技术栈
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- 无依赖，纯原生实现

## 使用说明

### 开发环境
直接使用浏览器打开HTML文件即可预览，建议使用Chrome浏览器的移动设备模拟器。

### 测试建议

4. 验证各个流程的完整性
